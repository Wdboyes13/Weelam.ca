AudiPlayer Source Building

Requirements

  • Qt5
  • Discord Social SDK (optional)1
  • Taglib
  • CMake
  • Git VCS
  • A C++ compiler (installed on most systems, prefferably Clang or GCC)

How to build

  1. Clone the repo git clone --depth 1 https://github.com/Wdboyes13/Audiplayer.git
  2. Change into the repo cd Audiplayer
  3. If you are buiding with discord, download and extract the Discord Social SDK into lib/discord_social_sdk
  4. Run CMake Configure cmake -B build
  5. Run CMake Build cmake --build build
    If you are building without discord, also add -DNODISCORD
  6. Install with CMake cmake --install build
  7. Audiplayer is now installed!

Notes

  1. The Discord Social SDK is optional because it needs to be requested directly from Discord
