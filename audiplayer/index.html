AudiPlayer

What is it?

AudiPlayer is a simple to use audio player focused on being as minimal as possible.

Features

  • Simple to use
  • Minimal
  • Supports MP3, FLAC, and WAV
  • Has Discord Integration
  • Cross-platform (Linux, macOS, Windows)
  • Utilizes audio file metadata
